# Alice（macOS）产品设计（V1-V3）

## 1. 产品定位
- 产品名称：Alice
- 首发平台：macOS
- 形态：菜单栏个人助手（Menu Bar App）
- 使命：帮助英文基础薄弱用户在任意应用中快速提取英文句子信息骨架（主谓宾）

## 2. 用户与场景
### 2.1 核心用户
- 职场用户：阅读英文邮件、文档、报告
- 学习用户：阅读英文教材、论文、新闻
- 泛阅读用户：在浏览器、聊天应用中处理长段英文

### 2.2 核心场景
- 浏览器新闻/博客阅读
- PDF 阅读器中的英文段落理解
- 邮件客户端中复杂英文长句速读
- IM/协作文档工具中的英文信息快速提炼

## 3. 价值指标
### 3.1 北极星指标
- 用户在不切换应用的前提下，5 秒内识别句子信息骨架

### 3.2 分层指标
- 一级：主谓宾解析准确率（优先）
- 二级：触发到结果展示的 p95 时延
- 三级：跨应用可用覆盖率、7 日留存、日均触发次数

## 4. 功能路线图
### V1（当前）
- 全局快捷键触发
- 悬停文本抓取（AX 优先 + OCR 回退）
- 自动分句并逐句输出主谓宾
- 低置信度提示与可选云端回退

### V2
- 复杂句增强：从句结构、并列结构、被动语态主干标注
- 代词指代提示（基础版）
- 输出结果可固定到侧边窗口便于对照阅读

### V3
- 学习闭环：历史记录、难句收藏、错题回看
- 个性化训练：根据用户弱项推荐句型练习
- 质量自适应：根据用户反馈动态调整本地/云端策略

## 5. 信息架构
### 5.1 菜单栏入口
- 运行状态
- 快捷键设置
- 权限状态（辅助功能、屏幕录制）
- 最近解析记录入口

### 5.2 全局悬浮结果卡
- 就近显示在光标附近
- 按句展示 S/V/O
- 展示来源应用与耗时
- 支持关闭与复制结果

### 5.3 设置页
- 快捷键管理
- 隐私设置（本地优先/允许云端回退）
- 权限诊断与修复引导

## 6. 系统架构（产品级）
### 6.1 输入层
- 全局快捷键监听
- 悬停上下文定位
- 文本抓取：AX API 优先，失败后 OCR 回退

### 6.2 语言层
- 段落分句
- 主谓宾解析（规则 + 小模型混合）
- 置信度计算
- 失败重试与云端回退策略

### 6.3 展示层
- 悬浮结果卡
- 错误提示与重试入口
- 回退状态标识（本地/云端）

### 6.4 数据层
- 本地配置：快捷键、隐私偏好
- 本地缓存：最近解析记录（可选）
- 匿名事件日志：触发、成功、失败、回退

### 6.5 策略层
- 模型路由策略：默认本地，低置信度才回退云端
- 隐私策略：最小化上传，默认不开启云端回退
- 质量策略：以准确率优先，时延次优

## 7. 隐私与合规
- 默认纯本地解析路径
- 云端回退需要用户显式开启
- OCR 截图仅内存短驻留，不落盘
- 只采集匿名事件，不采集用户账户身份信息

## 8. 非目标（当前阶段）
- 不做全文翻译
- 不做写作纠错
- 不做多语言互译平台

## 9. 发布策略
### 阶段 1：内部灰度
- 约 20 名种子用户
- 收集准确率与失败样本

### 阶段 2：公开 Beta
- 放量到约 500 用户
- 验证跨应用覆盖率与稳定性

### 阶段 3：正式发布
- 建立稳定质量看板（准确率、时延、失败类型）
- 进入 V2 能力迭代
